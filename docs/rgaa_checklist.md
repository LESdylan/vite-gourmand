# RGAA - Complete Accessibility Checklist

## 1. Images

### Criterion 1.1
- [ ] Does every image carrying information have a text alternative?
  - Criterion 1.1 tests and references

### Criterion 1.2
- [ ] Is every decorative image properly ignored by assistive technologies?
  - Criterion 1.2 tests and references

### Criterion 1.3
- [ ] For every image carrying information that has a text alternative, is that alternative relevant (except in specific cases)?
  - Criterion 1.3 tests and references

### Criterion 1.4
- [ ] For every image used as a CAPTCHA or test image with a text alternative, does that alternative identify the nature and function of the image?
  - Criterion 1.4 tests and references

### Criterion 1.5
- [ ] For every image used as a CAPTCHA, is there an alternative method to access the content or function of the CAPTCHA?
  - Criterion 1.5 tests and references

### Criterion 1.6
- [ ] Does every image carrying information have, if necessary, a detailed description?
  - Criterion 1.6 tests and references

### Criterion 1.7
- [ ] For every image carrying information that has a detailed description, is that description relevant?
  - Criterion 1.7 tests and references

### Criterion 1.8
- [ ] Every text image carrying information, in the absence of a replacement mechanism, should if possible be replaced by styled text. Is this rule respected (except in specific cases)?
  - Criterion 1.8 tests and references

### Criterion 1.9
- [ ] Is every image caption, if necessary, correctly linked to the corresponding image?
  - Criterion 1.9 tests and references
